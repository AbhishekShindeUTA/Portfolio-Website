Discretionary Access Control System
Using Web Based Front-End and MySQL

Implemented a DAC subsystem allowing for admin to assign and revoke specific privileges from a user over specific relations in a database. The Admin can select which permissions to be given to which user over which database relation. Admin can also update existing privileges. A user which creates a relation is the sole owner of that relation, unless they decide to give permissions over their relation to another user.

HTML5 CSS3 Bootstrap 4 PHP MySQL

Board Game Geek Review Classification
With Ridge Classifier and Flask Deployment

Using the BoardGameGekk reviews dataset, implemented a predictor to study and understand the behavior of data and to accurately predict a numerical rating value for a textual review for any game. The 13 million records dataset was cleaned, normalized and used to train the model to predict the rating. Smoothing techniques and probability calculations were used to generate a good accuracy. The project is deployed on a Live Server using Flask.

Flask Python Sklearn Numpy Pandas

Sentiment Analysis with Naive-Bayes Classifier
Using a Large Movie Reviews Dataset

Built a naive bayes classifier from scratch to classify the review of a movie into a positive or a negative review. The dataset consisted of 50,000 movies with each having a positive and a negative review. The model was tested using repeated K-fold method and the accuracy was improved using smoothing techniques and various text cleaning techniques.

Feasibility of Malware Packer Detection
Using Hardware Performance Counters

Research Objective was to analyze the Hardware Performance Counters of popular packing algorithms in order to classify families of packed algorithms in obfuscated malware programs. Goal was to accurately classify the packed algorithm using the HPC values to create a software signature. Model was designed using Naive-Bayes classifier to identify 3 results : LZMA Algorithm, UPX Algorithm, If the malware is packed with neither of the two algorithms.

Technologies used:

Python Sklearn Numpy Tkinter
